# How to Install the BrickVerse Client and Workshop

Installing the BrickVerse Client and Workshop is quick and easy. Follow these steps to get started:

1. **Go to the Download Page**\
   Open your web browser and navigate to:\
   [https://brickverse.gg/download](https://brickverse.gg/download)
2. **Download the Client**
   * On the download page, click the **Download** button for the BrickVerse Client.
   * Scroll down the page to review and agree to the End User License Agreement (EULA).
   * Once you agree, the download should start automatically.
3. **Download the Workshop**
   * On the same page, click the **Download** button for the BrickVerse Workshop.
   * Again, scroll down to agree to the EULA if prompted.
   * The download will begin automatically.
4. **Install Both Programs**
   * Once the downloads finish, open the installer files and follow the on-screen instructions to install the Client and Workshop.
5. **Launch BrickVerse**
   * After installation, open the BrickVerse Client to start playing or use the Workshop to create your own experiences.

**Tip:** Make sure your internet connection is stable during download to prevent interruptions.
